leuart_echo

This example loops back the leuart uart using interrupts. It waits in EM2 

How To Test:
1. Build the project and download to the Starter Kit
2. Connect UART device (such as CP210x) to STK
3. Type in terminal, lines will be echoed after enter is pressed, or 80
   characters are received 

Peripherals Used:
HFRCO  - 19 MHz
LFXO   - 32.768 KHz
LEUART0 - 9600 baud, 8-N-1
