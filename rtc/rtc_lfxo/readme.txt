rtc_lfxo
This project demonstrates the use of the RTC. The timer
is set to interrupt every 3 seconds and upon interrupt
LED1 is toggled.

How To Test:
1. Build the project and download to the Starter Kit
2. Observe LED1 turning on and off as the interrupts occur

Peripherals Used:
LFXO  - 32.768 kHz
RTC COMP0

Board:  Silicon Labs EFM32ZG Starter Kit (STK3200)
Device: EFM32ZG222F32
RTC - COMP0
PC11 - LED1

Board:  Silicon Labs EFM32HG Starter Kit (SLSTK3400A)
Device: EFM32HG322F64
RTC - COMP0
PF5 - LED1

Board:  Silicon Labs EFM32WG Starter Kit (STK3800)
Device: EFM32WG990F256
RTC - COMP0
PE3 - LED1

Board:  Silicon Labs EFM32GG Starter Kit (STK3700)
Device: EFM32GG990F1024
RTC - COMP0
PE3 - LED1

Board:  Silicon Labs EFM32LG Starter Kit (STK3600)
Device: EFM32LG990F256
RTC - COMP0
PE3 - LED1
